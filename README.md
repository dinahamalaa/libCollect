# libCollect

libCollect adalah aplikasi manajemen koleksi buku berbasis web yang dikembangkan menggunakan PHP dan MySQL. Aplikasi ini memungkinkan pengguna untuk menambah, melihat, mengubah, dan menghapus data buku melalui antarmuka yang sederhana dan mudah digunakan.

## Fitur

- Menampilkan daftar koleksi buku
- Menambahkan data buku baru
- Mengubah data buku
- Menghapus data buku
- Validasi input form
- Statistik sederhana

## Teknologi

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap

## Struktur Database

Database terdiri dari tabel koleksi buku yang menyimpan informasi seperti:
- Judul buku
- Penulis
- Penerbit
- Tahun terbit

## Cara Menjalankan

1. Clone repository
2. Pindahkan project ke folder `htdocs`
3. Buat database MySQL
4. Import file database `.sql`
5. Jalankan Apache dan MySQL melalui XAMPP
6. Akses aplikasi melalui browser
